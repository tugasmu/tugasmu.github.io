Sistem Pengumpulan Tugas Online

Platform modern untuk guru dan siswa dalam mengelola tugas sekolah dengan mudah dan efisien

Selamat Datang di Tugasmu

Sistem pengumpulan tugas modern yang dirancang khusus untuk kebutuhan sekolah dengan antarmuka yang intuitif.

Tentang Sistem

Tugasmu menyediakan solusi lengkap untuk manajemen tugas sekolah:

Untuk Developer

Kelola sistem secara keseluruhan, impor data siswa, dan pantau performa aplikasi.

Untuk Guru

Buat dan kelola tugas, beri nilai, pantau perkembangan siswa dengan dashboard lengkap.

Untuk Siswa

Akses tugas, kumpulkan pekerjaan, dan lihat nilai dengan mudah menggunakan ID unik.

Fitur Unggulan

Dashboard Real-time

Monitor aktivitas dan statistik secara langsung dengan update real-time.

Import Siswa

Impor data siswa dari file CSV dengan mudah, generate ID otomatis.

Buat Tugas

Buat tugas dengan deadline, instruksi detail, dan pengaturan nilai.

Kumpulkan Tugas

Siswa dapat mengumpulkan tugas dengan mudah menggunakan ID unik.

Laporan Detail

Analitik lengkap untuk guru dan administrator.

Keamanan

Sistem autentikasi aman dengan Supabase.
